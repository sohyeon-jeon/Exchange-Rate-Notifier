# Exchange-Rate-Notifier
환율 정보를 매매 기준율이 아닌 현찰 살 때의 기준으로 알림받고 싶어서 개발한 **환율알리미**입니다.  
텔레그램 봇을 활용하여 주기적으로 1시간마다 최신 환율 정보를 메시지로 받을 수 있습니다.

- 개발 환경  
  python 3.8.10

- 필요 라이브러리  
bs4 == 0.0.1  
python-telegram-bot ==13.14
